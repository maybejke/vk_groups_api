**Parse VK Groups**

Зависимости:

_install requirements.txt_

Скрипт запускается:

`python vk_parse.py`

Изменение параметров(введены тестовые): 

`SERVICE_TOKEN` - токен(введен мой, можно тестить с ним)
 
`group_id` - вводим айди группы в вк

**API запросы:** 
`https://vk.com/dev/groups.getMembers`
